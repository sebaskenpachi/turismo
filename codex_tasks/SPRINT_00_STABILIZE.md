# Tarea Codex — Sprint 0: estabilizar proyecto Flutter

Revisa el proyecto Flutter actual de Rutas Panamá.

Objetivo:
Que el proyecto compile correctamente en Android y quede listo para implementar GPS/Google Maps.

Instrucciones:
1. Si el proyecto no fue creado con `flutter create`, créalo correctamente sin perder la carpeta `lib/` actual.
2. Revisa `pubspec.yaml` y actualiza dependencias compatibles.
3. Agrega `.gitignore` apropiado para Flutter.
4. Ejecuta `flutter pub get`.
5. Ejecuta análisis estático si Flutter está disponible.
6. Corrige errores de compilación básicos.
7. No agregues Firebase, Supabase ni Google Places todavía.
8. No quemes API keys.

Entregable:
- Proyecto compila en Android.
- Resumen de archivos modificados.
- Comandos exactos para correr la app.
- Lista de pendientes técnicos.
